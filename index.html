ABOUT

Cryptocurrency Trading Dashboard

The purpose of this project is to provide a static representation of the data visualizations that will be used for the VolaTrade cryptocurrency trading dashboard. I will provide 3 different visualizations for this project. First, a candlestick chart which shows the chosen crypto asset's open, high, low, and close price for a given time period. Second, a depth chart will show the amount of buy and sell orders for a particular asset at varied prices. Third a heat map will show the performance of each asset for a given time period.
